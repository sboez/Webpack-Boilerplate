# :package: Webpack-Boilerplate

<img src="https://img.shields.io/badge/webpack-5.52.1-blue">

My basic Webpack boilerplate with Babel and Sass

# Usage

```shell
git clone https://github.com/sboez/Webpack-Boilerplate.git
```

```shell
npm install
```

## Development

```shell
npm run dev
```

## Production

```shell
npm run build
```
